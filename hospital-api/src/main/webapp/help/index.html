登录
启动APP，在启动之后显示登录页：
登录的用户名和密码由医保经办机构分配。

首页

认证统计
点击首页-认证统计，进入认证统计页：
默认是当月1日开始统计，也可以选择日期
进行查询。

住院信息
点击首页-住院信息，进入住院信息页：

可根据姓名/身份证号/床号/住院号关键字、科室、状态、
开始时间、结束时间进行高级查询。点击“查看详情”，进入病人详细信息页

该页可查看病人的基本信息，以及历史抽查记录，点击
"查看"，进入抽查认证详情页：

实人认证
点击首页-实人认证，进入实人认证页：

可选择科室、输入姓名/身份证号/床号/住院号关键字进
行查询，显示病人基本信息，点击右侧图标进行拍照，拍照后后台进行认证，并且显示认证结果：

如果认证失败，可选择重新认证，或点击我要申诉：

选择申诉理由，填写事由说明，点击提交，后台进行申
诉审核。如果病人脸部受伤，可选择将该病人标记
为特殊人员

认证记录
点击首页-认证记录，进入认证记录页：

默认是当月1日开始查询，也可以选择日期进行查询。可点击“查看详情”按钮，进入记录详情页：
可选择病人当前状态或输入姓名/身份证号/床号/住院号关键字进行查询。

常见问题
点击首页-常见问题，进入常见问题页：

消息
点击消息，进入消息页：
消息提示抽查任务，审核结果等。可点击"查看详情"，
查看抽查认证详情。

我的
点击我的，进入
点击我的-关于，查看APP相关信息
点击我的-帮助，查看APP帮助文档
点击我的-清除缓存，清除APP缓存
点击退出当前账号- 进入登录页

修改密码
点击我的-修改密码，进入：输入原密码、新密码、确认密码后，点击修改密码，即可完成密码修改操作。
